# Plan 07-02: Agent prompt injection + cache key invalidation

## Objective

Make Plan generation aware of stored lab rules. Today, every `/api/plan` request hashes only the hypothesis text — meaning Plan A (zero rules) and Plan B (one rule captured) collide on the same cache entry and Plan B silently returns Plan A. The propagation demo dies at that step.

This plan:

1. Renames `hashHypothesis` → `hashRunInput(hypothesis, labRules)` in `src/lib/plan/cache.ts` so the cache key changes whenever the rule set changes (D7-11).
2. Threads `getLabRules()` through `src/app/api/plan/route.ts` and passes `labRules: LabRule[]` to all 4 agent runners.
3. Extends each of the 4 agent runners (researcher, skeptic, operator, compliance) to accept an optional `labRules` arg and append a `LAB RULES (apply these to your output):` block to the user prompt when non-empty. System prompts stay UNCHANGED (preserves Google's implicit prefix cache).
4. Gives the compliance agent extra rule-awareness: when a rule has scope `validation_check` or `global`, instruct the agent to emit a `compliance_notes[]` entry referencing the rule's source text — so the propagation lands VISIBLY in the UI, not only in agent reasoning (D7-10).

This plan modifies **6 existing files** and creates none. It depends on Wave 1 (07-01) for `getLabRules` and the `LabRule` type. It is **safe to run in parallel with Wave 3** (07-03) — the file sets do not overlap.

```ts
/**
 * D7-11: hash the run input — hypothesis text + the sorted IDs of any
 * lab rules in effect. This guarantees that Plan A (no rules) and Plan B
 * (one or more rules captured) produce DIFFERENT cache keys, so Plan B
 * actually re-runs the agents instead of short-circuiting to the cached
 * Plan A. Empty rules array hashes identically to legacy hashHypothesis.
 */
export async function hashRunInput(
  hypothesis: string,
  labRules: LabRule[] = [],
): Promise<string> {
  const normalized = hypothesis.trim().toLowerCase();
  const ruleKey = JSON.stringify(
    labRules.map((r) => r.id).sort((a, b) => a.localeCompare(b)),
  );
  const buf = await crypto.subtle.digest(
    "SHA-256",
    new TextEncoder().encode(`${normalized}|${ruleKey}`),
  );
  return Array.from(new Uint8Array(buf))
    .map((b) => b.toString(16).padStart(2, "0"))
    .join("");
}

/**
 * Legacy alias. Kept so any forgotten caller compiles. Same hash as
 * hashRunInput(input, []) — guarantees backwards compatibility for
 * cached Plan A entries written before Phase 7.
 */
export async function hashHypothesis(input: string): Promise<string> {
  return hashRunInput(input, []);
}
```
