## Critical Anti-Patterns (do NOT repeat)

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|----------------------|
| Vercel preset = "Other" trap | Creating the Vercel project before `package.json` lands on GitHub auto-detects Framework Preset as "Other" instead of Next.js — every route returns NOT_FOUND | advisory | If standing up a new Vercel project for a new sub-app, always push the scaffold first, then verify Framework Preset = correct framework after import |
